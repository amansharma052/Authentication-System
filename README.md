# 🔐 Authentication System Backend

A secure **Authentication System Backend** built using **Node.js, Express.js, MongoDB, and JWT**.
This project provides **user registration and login APIs** with password hashing and token-based authentication.

---

## 🚀 Features

* User Registration
* User Login
* Password Hashing using bcrypt
* JWT Token Authentication
* Secure REST APIs
* MongoDB Database Integration

---

## 🛠 Tech Stack

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT (JSON Web Token)
* bcryptjs
* dotenv

---

## 🔑 Authentication Flow

1. User registers with email and password
2. Password is hashed using bcrypt
3. User logs in with credentials
4. Server verifies password
5. JWT token is generated and returned
